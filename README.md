# VPSService_bingimg

自2019年9月26日至今的必应每日美图爬虫。

下载路径在img_list中。

每日自动更新。


如果需要部署，记得添加一个telegram的配置文件。


tg_config.json
```
{
  "token": "139xxxxxxxxxxx9:xxxxxxxxxxxxxxxxxxxxxxxxxxx-XZhHuU", #<---bot token
  "chat_id": "@bing_daily_image"                                 #<---chat_id,可以是频道/群组id，也可以是频道/群组的username
}

```
